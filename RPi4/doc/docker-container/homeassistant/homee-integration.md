# Homee-Integration
Es gibt zwei Möglichkeiten Homee und Home Assistant zu integrieren: **homee2mqtt** und **Home Assistant homee integration**.

## Home Assistant homee integration
Vorteil dieser Integration ist, dass sie _direkt_ (ohne den _Umweg_ über mqtt) erfolgt und dass es aus Home Assistant-Sicht lesend und schreibend funktioniert.  
Die Installation erfolgt über HACS, die Konfiguration über Einstellungen / Integrationen / + INTEGRATION HINZUFÜGEN / Homee.
